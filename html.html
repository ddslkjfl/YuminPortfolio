학과 홈페이지 개요

디지털 마케팅과 미디어 콘텐츠 역량을 집중적으로 보여주는 포트폴리오 웹사이트입니다.
최신 트렌드와 실무 중심의 디자인, UX/UI를 통해 전문성과 시각적 완성도를 강조합니다.

  • 역할: 디지털 마케팅 및 디자인 역량 시각화
  • 툴: Visual Studio Code
  • 주요 기능: 프로젝트 소개, 결과물 전시, 반응형 웹 지원

웹사이트 제작

학과 홈페이지 리뉴얼

미니멀하고 트렌디한 디자인으로
학과의 정체성을 시각적으로
표현하며, 학생과 방문객에게
정보 탐색의 즐거움과 신뢰감을
제공합니다.

이미지를 누르면 홈페이지로 이동하여 볼 수 있습니다.

디자인 포인트

  • 일관된 컬러와 폰트로 브랜드 아이덴티티를 명확히 전달하는 시각적 통일성 유지
  • 여백과 구성미를 살린 미니멀 디자인으로 콘텐츠 중심의 전문성 강조
  • 모든 기기에서 최적화된 경험을 제공하는 반응형·모바일 친화적 웹디자인 구현
  • 마우스 오버, 트랜지션 등의 인터랙션 요소로 사용자 참여와 몰입 유도
  • 고해상도 이미지와 영상으로 브랜드의 품질감과 시각적 완성도 강화
